    MANRS Observatory FAQ

    Home MANRS Observatory MANRS Observatory FAQ

    Where does the data come from?

    This site uses publicly available data in order to measure routing activities on the Internet. A description of each is available on the Data Sources page.

    How is data collected?

    A separate data collection system gathers data from multiple external sources, calculates the metrics needed to measure MANRS Readiness from these external sources, and stores them in a database. These snapshots are produced every month.

    The dashboard allows one to look at these data using a user-friendly graphical interface. 

    What is the quality of the reporting metrics?

    The MANRS Observatory uses passive measurements and existing databases from trusted data providers to measure MANRS Readiness for all networks on the Internet. Sometimes this data is incomplete or contains false positives (when an event is wrongly marked as a security incident), or false negatives (when a security incident goes under the radar).

    Overall, the measurement framework produces a credible picture but is not 100% accurate. We are continuously working on improving the data quality as well as adding additional data sources.

    How often is the data refreshed?

    A monthly snapshot is computed and published at the beginning of the following month.

    What can I expect to see?

    The Overview tab shows the number of incidents, networks involved, and quality of published routing information in the IRR and RPKI in the selected region and time period. By default, it’s a global view of the last month.

    Learn how to filter information

    You can also see statistics for specific economies by hovering anywhere over the global map. The size of the dot in a country or region represents the number of networks in that particular economy. The data in the State of Routing Security and MANRS Readiness will reflect the chosen country.

    The History tab displays a rolling 12-month period. The MANRS Observatory started collecting data in January 2019 and currently only shows that month forward.

    The Comparison tab shows the difference between a reference and options chosen through filters.

    MANRS participants can also access a historical overview of a specific member ASN, per MANRS Action, as well as detailed information related to a specific metric via their Member login.
