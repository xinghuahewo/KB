# BGP 知识库制品策略

Git 只保存源码、配置、测试、制品定位信息和小型文本说明。以下运行数据必须通过服务器制品库交付：原始语料、清洗语料、chunk、向量索引、SQLite 知识库、派生数据与生成报告。

当前制品版本为 `2026-07-10-93a4c97`，存储于：

```text
/srv/bgpkb/artifacts/releases/2026-07-10-93a4c97/
```

制品包含 `data/` 目录和 `SHA256SUMS`。部署或恢复时，在服务器执行：

```bash
cd /srv/bgpkb/artifacts/releases/2026-07-10-93a4c97
sha256sum -c SHA256SUMS
```

校验通过后，再将该版本的 `data/` 同步到运行副本。发布新制品时使用新的不可变版本目录，并记录清单 SHA-256、文件数量与生成提交号；不得覆盖旧版本。
